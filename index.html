PV-PP Grenade Self-Sacrifice Benchmark

This repository hosts a benchmark scenario used to compare decision systems under high-stakes sacrificial conditions.

The benchmark examines whether competing frameworks can recognize, justify, or reject self-sacrifice when an agent can save others at personal cost.

Benchmark Focus

  • Self-sacrifice vs self-preservation
  • Mission success under lethal threat
  • Multi-agent survival outcomes
  • Scalar utility vs structured decision models
  • PV-PP comparative performance

Repository Contents

  • Simulation code
  • Scenario definitions
  • Output runs and test cases
  • Research notes
  • Future benchmark expansions
